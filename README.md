## How to Run the Program

1. **Clone the repository:**
    ```sh
    git clone https://github.com/HermanPlay/se2-2025-tdd.git
    cd se2-2025-tdd
    ```

2. **Install dependencies:**
    ```sh
    go mod tidy
    ```

3. **Run the program:**
    ```sh
    go run cmd/app/main.go
    ```

4. **Run tests:**
    ```sh
    go test ./...
    ```
